# Ayamine Ikam - Portfolio

## 🚀 Modern Corporate Portfolio

Refonte stratégique du portfolio personnel avec design high-end inspiré de [djanatocarlos.com](https://djanatocarlos.com/).

### ✨ Features
- **Design**: Blanc immaculé, accents rouge royal (#E63946), typographie Inter/Poppins
- **Responsive**: Mobile-first, navigation pill burger
- **Animations**: Framer Motion fade-in-up scroll-triggered
- **Interactions**: WhatsApp direct (+250723672942), GitHub stars CTAs
- **Stack**: React 19 + TailwindCSS v4 + Vite

### 📱 Quick Start
```bash
npm install
npm run dev
```

### 📂 Structure
```
src/
├── App.jsx (layout)
├── index.css (Tailwind @theme)
├── components/
│   ├── Navbar.jsx (pill nav SVG icons)
│   ├── Hero.jsx (badge + CTAs rounded-full)
│   ├── About.jsx (highlights cards)
│   ├── Services.jsx (services grid)
│   ├── Projects.jsx (cards GitHub links)
│   ├── Contact.jsx (SVG icons WhatsApp)
│   └── Footer.jsx (gradient dark)
```
