## 💾 Installation

### Prerequisites
- **Node.js** v16 or higher
- **npm** or **yarn** (package manager)

### Steps

1. **Clone the repository**
```bash
git clone https://github.com/votre-username/my-portfolio.git
cd my-portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Start the development server**
```bash
npm run dev
```

The site will automatically open at `http://localhost:5173/`

## 🚀 Usage

### Development mode
```bash
npm run dev
```
Starts the development server with Hot Module Replacement (HMR) enabled. Changes are applied in real-time.

### Code linting
```bash
npm run lint
```
Checks code quality and applies ESLint rules.

### Production build
```bash
npm run build
```
Creates an optimized and minified version ready for deployment.

### Preview build
```bash
npm run preview
```
Shows a local preview of the compiled version before deployment.
